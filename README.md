# c_atimer_lib
自研LinuxC高精度 异步定时器+线程池

## 项目整体分析
### 一、项目架构概览
┌─────────────────────────────────────────────────────────────────┐
│                        用户应用程序                              │
│   atimer_init → atimer_register → atimer_reset/cancel → destroy │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                    atimer.c (定时器管理器)                        │
│  ┌─────────────┐    ┌──────────────┐    ┌─────────────────┐     │
│  │ 管理线程     │───▶│ epoll 监听   │───▶│ timerfd 事件    │     │
│  │ (SCHED_FIFO)│    │ (ET模式)     │    │ (纳秒级精度)    │     │
│  └─────────────┘    └──────────────┘    └─────────────────┘     │
│         │                                           │           │
│         │ 提交任务                                   │           │
│         ▼                                           │           │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                  tpool.c (线程池)                        │   │
│  │  ┌────────────┐  ┌────────────┐  ┌────────────┐         │   │
│  │  │ Worker #1  │  │ Worker #2  │  │ Worker #N  │         │   │
│  │  └────────────┘  └────────────┘  └────────────┘         │   │
│  │              环形任务队列 (避免内存碎片)                   │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐

└─────────────────────────────────────────────────────────────────┘
